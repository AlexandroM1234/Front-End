# How-To Front-End

## About

- The App is a simple how to app where you can log in and signup and start creating how to guides for yourself and other users

## Install

    $ git clone https://github.com/How-To2/Front-End.git
    $ cd how-to
    $ npm install

## Start

    $ npm start

## TechStack

- React
- Styled Components
- Redux
- Axios
- Jest
